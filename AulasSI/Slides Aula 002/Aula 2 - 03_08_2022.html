Como usar estado
Conceitos importantes
Até agora vimios widgets sem estado
Uma vez compilados, seu estado não mudará mais durante toda a aplicação
Mas, precisamos interagir com o usuário? isso faz mudar dados da interface
State guarda o estado da aplicação
O Flutter se parece com React nesse sentido
Por si só, widgets não mudam, eles são imutáveis, o que mudam são os estados, dando muito mais flexibilidade a esse componentes

Começando nosso App
Usando StatelessWidget e StatefulWidget
StatelessWidget: Um widget que não guarda estado
StatefulWidget: Um widget que possui um estado
import 'package:flutter/material.dart'; void main() => (runApp(MyApp())); class MyApp extends StatelessWidget{ @override Widget build(BuildContext context) { return MaterialApp( title: "Chamados", home: Scaffold( appBar: AppBar( title: Text("Chamados"), ), body: Center( child: Text("Cadastro de chamados"), ), ), ); } }
Dart
Para ter um widget sem estado, é preciso somente
Criar uma classe que estenda StatelessWidget.
Sobrescrever o metodo build()
Retorar o widget criado
Vamos tunar esse widget
Vamos herdar de StatefulWidget
class MyApp extends StatefulWidget{...}
Dart
O QUE ACONTECEU???
Para ter um widget com estado, é preciso somente
possuir o metodo createState()
Não precisamos do build no Widget
Quem retorna o build() é o estado
Continuando o conceito
Para não esquecer
void initState() { // ... super.initState(); }
Dart
Método que pode ser sobrescrito na classe State para inicializar o estado
setState(() { //... });
Dart

Componentes Stateless gerenciado por Stateful?
Quanto mais componentes Stateless na nossa aplicação, melhor
Inevitavelmente, precisamos gerenciar estado, por isso, separamos nossos componentes para que somente tenhamos filhos Stateless dentro dos componentes Stateful
Como podemos fazer isso
Crie uma classe CustomTextField
lib/widgets/custom_text_field.dart
import 'package:flutter/widgets.dart'; import 'package:flutter/material.dart'; class CustomTexField extends StatelessWidget { final String hintText; final String label; final int lines; final void Function(String) onChanged; const CustomTexField( {required this.hintText, required this.label, this.lines = 1, required this.onChanged}); @override Widget build(BuildContext context) { return Padding( padding: const EdgeInsets.all(8.0), child: TextField( maxLines: this.lines, minLines: this.lines, keyboardType: TextInputType.multiline, decoration: InputDecoration(hintText: this.hintText, label: Text(this.label)), onChanged: onChanged), ); } }
